# Privacy Policy

## Pflegedeutsch – German for Healthcare Learners

Last Updated: July 2026

Pflegedeutsch stores learning progress, lesson completion status, quiz scores, writing assessment scores, and final examination results to provide course continuity and progress tracking.

### Information We Store

- Telegram User ID
- Lesson progress
- Quiz scores
- Writing exercise scores
- Final assessment results

### How We Use Information

The information collected is used solely to:

- Track learning progress
- Save assessment results
- Generate course completion statistics
- Improve the learning experience

### Information Sharing

Pflegedeutsch does not sell, rent, or share user data with third parties.

### Data Security

Reasonable measures are taken to protect stored learning data and user progress information.

### User Rights

Users may stop using the bot at any time. Stored learning records may be removed upon request to the bot administrator.

### Contact

Pflegedeutsch is developed and operated by BigMic EduTech.

For questions regarding this Privacy Policy, please contact BigMic EduTech through Telegram.
